# Spring Boot Hello Example

### 개요

Spring Boot를 사용하여 간단한 웹 애플리케이션을 빌드하고, 몇 가지 유용한 서비스를 추가합니다.

### 애플리케이션 실행

애플리케이션을 실행하려면 터미널 창에서 프로젝트의 루트 디렉토리로 이동하고 다음 명령을 실행합니다.

```shell
./gradlew bootRun
```

Maven을 사용할 경우:

```shell
./mvnw spring-boot:run
```

### 실행 중지

다른 터미널 창에서 다음 명령어를 입력하면 실행 중지됩니다.

```shell
./gradlew -stop
```
